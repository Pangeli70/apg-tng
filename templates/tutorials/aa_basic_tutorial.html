Step 2: Partial template

The following html code shows how to:

  • use field interpolation inside the user_details "partial"
  • use native javascript code inside the html template
    <% tutorial.partial %>

Step 3: js data

The following js data is used to interpolate fields in html templates. 

    <% tutorial.data %>

Step 4: Drash resource code

The following ts code is used to render the page template and interpolate the templateData.

    <% tutorial.resource %>
